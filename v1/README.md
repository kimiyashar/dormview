# DormView v1 (archived)

This folder is the **original v1 build**: E-Tower **Traditional Double only**, before the
Single and Semi-Suite Triple room types were added.

- `index.html` - open it directly in a browser (same as v2, needs internet on first load).
- It is frozen. New work happens on v2 (the root of this repo), which is the build live at
  https://dormview.vercel.app.
- Its autosave key is `dormview_v1_archive` (renamed from the shared v2 key) so a saved v1
  layout can never overwrite a v2 layout when both are served from the same domain.

Source: this is the exact code at git tag `v1` (commit 9749e08), with only the autosave key
renamed.
